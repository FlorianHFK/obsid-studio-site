# 🤝 Code de Conduite

## Notre Engagement

Dans l'intérêt de favoriser un environnement **ouvert et accueillant**, nous, en tant que contributeurs et mainteneurs, nous engageons à faire de la participation à notre projet et à notre communauté une expérience **exempte de harcèlement** pour tout le monde, indépendamment de l'âge, du corps, de la capacité, de l'ethnie, de l'identité de genre et de son expression, du niveau d'expérience, de l'éducation, de la socio-économie, de la nationalité, de l'apparence personnelle, de la race, de la religion ou de l'identité et de l'orientation sexuelle.

## Nos Standards

Les exemples de comportements qui contribuent à créer un environnement positif pour notre communauté incluent :

- Utiliser un **langage accueillant et inclusif**
- Respecter les **différents points de vue et expériences**
- Accepter les **critiques constructives** avec grâce
- Se concentrer sur **ce qui est meilleur pour la communauté**
- Faire preuve d'**empathie** envers les autres membres de la communauté

Les exemples de comportements **inacceptables** incluent :

- L'utilisation de **langage ou d'images sexistes, racistes, homophobes, transphobes, ableistes ou autres formes de langage discriminatoire**
- Le **harcèlement** sous toutes ses formes
- Les **attaques personnelles** ou les insultes
- La **diffamation** publique ou privée
- La publication d'**informations privées** d'autrui sans consentement explicite
- Toute autre conduite qui pourrait **raisonnablement être considérée comme inappropriée** dans un cadre professionnel

## Nos Responsabilités

Les mainteneurs du projet se réservent le droit de **supprimer, modifier ou rejeter** tout commentaire, commit, code, modification, issue ou autre contribution qui ne respecte pas ce Code de Conduite. Nous nous réservons également le droit de **bannir temporairement ou définitivement** tout contributeur pour des comportements que nous jugeons inappropriés, menaçants, offensants ou nuisibles.

## Portée

Ce Code de Conduite s'applique à tous les **espaces du projet**, y compris, mais sans s'y limiter :

- Le code et les commentaires dans le dépôt GitHub
- Les issues et les pull requests
- Les discussions sur GitHub
- Les canaux de communication (Discord, Slack, etc.)
- Les événements en ligne ou hors ligne

## Application

Les cas de comportement abusif, harcelant ou inacceptable peuvent être signalés en contactant **Florian HFK** (`hachefk23@gmail.com` ou [@FlorianHFK](https://twitter.com/FlorianHFK)). Toutes les plaintes seront **examinées et enquêtées** et donneront lieu à une réponse qui sera jugée nécessaire et appropriée aux circonstances.

Les mainteneurs du projet sont tenus de **respecter la confidentialité** concernant le signalement de tout incident.
