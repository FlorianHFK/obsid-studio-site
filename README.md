## Vision

**OBSID-Studio** is a premium local-first AI creative OS.

It is not just a generator. It is the product layer above a complex local AI stack: create, predict, assist, diagnose, organize, reuse and ship.

<p align="center"><img src="assets/real-studio-preview.svg" alt="OBSID-Studio preview" width="100%"></p>

## What it does

| Create | Assist | Organize |
|---|---|---|
| Image, video, music, voice, text | Project guidance, prompts, workflows, next best action | Outputs, proofs, packs, reusable memory |
| Generator Store | Local agents | Engine Health and guided diagnostics |
| Creative packs | Predictive suggestions | Runtime clarity, no fake READY |

## Product architecture

<p align="center"><img src="assets/readme-product-map.svg" alt="OBSID product map" width="100%"></p>

OBSID-Studio is NEXUS-backed, not NEXUS-brut: NEXUS remains the forge, OBSITOOLS remains the heavy operator cockpit, and OBSID-Studio becomes the clean product experience.

## ORBYSS

ORBYSS is the living visual core of the studio.

It represents the state of the product: idle, thinking, generating, blocked, diagnosis, recovery and export.

Target identity: asymmetrical, crystalline, biomechanical, luminous, reactive and premium.
