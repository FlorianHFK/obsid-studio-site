# OBSID-Studio

Product preview for a local-first creative studio.

Live site: https://florianhfk.github.io/obsid-studio-site/

## Vision

OBSID-Studio turns local AI engines into a premium creative cockpit.

It is designed for image, video, music, voice, text, workflows, outputs, project memory and local agents.

## ORBYSS

ORBYSS is the living visual core of the studio.

It represents system states such as idle, thinking, generating, repair and export.

## Product promise

Make local AI clearer, cleaner, more visual, more reusable and more sellable.
