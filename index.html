Quelle configuration est requise ?

Minimum : Windows 10/11 x64, 16 Go RAM, GPU dédié.
Recommandé : Windows 11 x64, 32 Go RAM, GPU NVIDIA 8 Go+ (pour la génération d'images/vidéo).

Mes données restent-elles locales ?

Oui, 100%. Aucune donnée ne quitte votre machine. Pas de télémétrie, pas de tracking, pas de cloud caché. OBSID-Studio est conçu pour le local-first.

Quelle est la différence avec les IA cloud ?

Avec le cloud, vous payez à l'usage, vos données sont sur des serveurs distants, et vous dépendez de la connexion. Avec OBSID-Studio : un paiement unique, tout sur votre PC, illimité.
